Carpeta de imágenes del Manual (ManualScreen)

Cómo nombrar las imágenes
-------------------------
En el archivo `assets/manual_es.txt` hay marcadores como:
  [Imagen 12: Importar]

Convención simplificada:
La app ignora el texto después del número y busca SOLO por número:

  "[Imagen 12: Importar]"  ->  "Imagen_12"

Guarda tus imágenes con alguno de estos nombres/extensiones:
  - Imagen_12.png
  - Imagen_12.jpg
  - Imagen_12.jpeg
  - Imagen_12.webp

Ejemplos:
  - Imagen_1.png
  - Imagen_4.jpg

Si la app no encuentra una imagen, mostrará un texto "Imagen no encontrada".
